# Product Overview

This is a Flutter-based cross-platform mobile application with social/communication features. The app supports iOS, Android, Web, macOS, Linux, and Windows platforms.

## Core Features

- **Authentication System**: Complete user auth flow including sign up, sign in, password recovery, and password change
- **User Profiles**: Profile management with image upload and editing capabilities
- **Real-time Messaging**: Chat functionality with Socket.IO for real-time communication
- **Notifications**: Local push notification support
- **Settings & Legal**: User settings, privacy policy, and terms of service screens

## Design Approach

- Mobile-first responsive design using ScreenUtil (design size: 428x926)
- Material Design with custom theming
- Google Fonts for typography
- Reusable component library for consistent UI
